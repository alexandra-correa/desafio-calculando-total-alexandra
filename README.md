# Calculando el Total - JavaScript

Proyecto desarrollado como parte del módulo de JavaScript. El objetivo de este desafío es practicar la manipulación del DOM, el uso de eventos y la actualización dinámica de contenido utilizando JavaScript.

---

## Descripción

La aplicación permite calcular el total a pagar de un producto en función de la cantidad seleccionada. El usuario puede aumentar o disminuir la cantidad mediante botones y el total se actualiza automáticamente.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- DOM
- Eventos

---

## Funcionalidades

- Incrementar la cantidad de productos.
- Disminuir la cantidad de productos.
- Evitar cantidades negativas.
- Calcular automáticamente el total a pagar.
- Actualizar dinámicamente el contenido en pantalla.

---

## Conceptos aplicados

- `querySelector()`
- `innerHTML`
- `addEventListener()`
- Variables
- Condicionales
- Manipulación del DOM
